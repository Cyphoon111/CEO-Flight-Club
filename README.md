# CEO Flight Club

Splash page for **CEO Flight Club — By Design**, *An Exclusive National Network of Mid-Market CEOs & C-Suite Leaders*.

A single-page, fully responsive splash featuring the gold metallic logo centered on a black canvas with an elegant gold-gradient tagline.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Page markup |
| `styles.css` | Styling (responsive, print-aware) |
| `assets/logo_hero.png` | Centered gold logo mark |
| `assets/splash.png` | Favicon |

## Run locally

Open `index.html` in any browser, or serve the folder:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```

## Deploy with GitHub Pages

1. Go to **Settings → Pages**
2. Under **Build and deployment → Source**, choose **Deploy from a branch**
3. Select branch **`main`** and folder **`/ (root)`**, then **Save**
4. Your site will publish at `https://cyphoon111.github.io/CEO-Flight-Club/`
